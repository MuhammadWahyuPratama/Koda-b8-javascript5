# Javascript 4

## Built-in methods

Cara kerja Program dengan built-in method dan tidak menggunakan built-ijn method

-Manual
prosesnya dengan cara menggunakan for untuk melooping data di dalam array satu persatu di cek memakai indx , setelah itu program mengecek apakah angka tersebut ganjil atau genap , lalu menampilkan gasil dan mengulang lagi sampai data di dalam array habis

- Menggunakan built in method
  prosesnya dimulai dari method forEach() yang otomatis melakukan perulangan pada array. lalu setiap isi array ambil satu per satu melalui parameter(item) ,lalu di cek angka ganjil atau genap , setelah itu hasil di tampilkan setelah semua proses selesai.
